# 前端页面说明

## 1. 登录页 / 注册页
- 表单标签完整
- 登录失败与注册失败反馈明确

## 2. 首页
### 用户首页
- 借阅摘要
- 推荐图书
- 热门/新书
- 最近借阅

### 管理首页
- 核心统计
- 当前待办
- 库存紧张图书
- 最近新增图书
- 推荐统计摘要

## 3. 图书页
### 用户侧
- 关键词检索
- 分类号筛选
- 卡片化图书列表
- 图书详情弹层
- 相似图书入口

### 管理侧
- 表格管理
- 批量改位置
- 批量删除
- 新增/编辑图书
- 上传封面

## 4. 借阅页
### 用户侧
- 全部 / 待审核 / 待取书 / 借阅中 / 历史记录
- 状态卡片化展示
- 续借、归还、撤销、确认取书

### 管理侧
- 待审核 / 待取书 / 借阅中 / 逾期 / 全部
- 审核、驳回、撤销通过、代归还

## 5. 推荐页
### 用户侧
- 推荐卡片
- 推荐理由
- 换一批
- 与图书详情联动

### 管理侧
- 推荐总数
- 命中数
- 命中率
- 覆盖用户
- Top 图书命中表现

## 6. 个人中心 / 用户管理 / 分类管理
- 个人中心分区
- 用户管理支持检索与角色调整
- 分类管理轻量维护
