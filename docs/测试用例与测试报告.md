# 测试用例与测试报告

## 1. 测试范围
- 认证流程
- 用户资料
- 图书管理
- 分类管理
- 借阅流程
- 推荐流程
- 首页聚合接口
- 图书详情与相似图书

## 2. 关键用例
### 2.1 认证
1. 正确账号密码登录成功
2. 错误密码登录失败并返回提示
3. 新用户注册后可直接登录

### 2.2 图书
1. 关键词检索返回匹配结果
2. 管理员新增图书成功
3. 管理员编辑图书成功
4. 管理员上传封面成功
5. 打开图书详情时返回相似图书

### 2.3 借阅
1. 用户发起借阅成功
2. 管理员审核通过成功
3. 用户确认取书成功
4. 用户续借成功
5. 用户归还成功

### 2.4 推荐
1. 用户查看推荐成功
2. 用户刷新推荐成功
3. 管理员查看推荐统计成功

### 2.5 首页
1. 用户访问 `/api/dashboard/overview` 返回个人摘要
2. 管理员访问 `/api/dashboard/overview` 返回管理摘要
