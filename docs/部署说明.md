# 部署说明

## 1. 环境要求
- JDK 21
- Maven 3.9+
- Node.js 18+
- MySQL 8.x

## 2. 数据库
1. 创建数据库 `library_system`
2. 导入 `sql/library_system.sql`
3. 需要测试数据时可导入 `sql/seed_100_test_data.sql`

## 3. 后端启动
```bash
cd backend
mvn spring-boot:run
```
默认端口：`8081`

## 4. 前端启动
```bash
cd frontend
npm install
npm run dev
```

## 5. 生产构建
```bash
cd frontend
npm run build
```

## 6. 关键配置
- 数据库连接：`backend/src/main/resources/application.yml`
- JWT 密钥：`security.jwt.secret`
- DeepSeek API Key：环境变量 `DEEPSEEK_API_KEY`
- 推荐远程评分：`RECOMMEND_SCORE_API_URL`、`RECOMMEND_SCORE_API_KEY`

## 7. 封面上传
- 默认保存到 `backend/uploads/book-covers`
- 可通过 `file.upload-dir` 调整路径
